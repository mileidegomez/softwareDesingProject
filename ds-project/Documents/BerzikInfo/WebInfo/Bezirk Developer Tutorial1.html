Tutorial 1: Hello World

Overview

In this tutorial, you'll create a simple Hello World Zirk that uses Bezirk to publish a Hello World message. To reiterate, a Zirk is any software that uses Bezirk to communicate.

Note
If you're not familiar with setting up an Android Studio project and using Gradle, please see the previous tutorial: Project Setup.

Learning Objectives

At the end of the tutorial, you'll be able to:
  • Create an Event with custom message text for publishing
  • Create a Zirk that registers with Bezirk and publishes this event

Create a Custom Event To Be Sent

Bezirk uses the Publish-Subscribe communication style, i.e., Zirks can send events by publishing them, and other Zirks can subscribe to receive these events. These custom events, which typically contain custom variables, are created by subclassing the Event class. So let's first create our custom Event.

  1. In the java folder under src\main\ of your project, create a new Java class called "HelloWorldEvent".
  2. Import the Bezirk Event class and extend it to add a variable, helloText 
    import com.bezirk.middleware.messages.Event;

public class HelloWorldEvent extends Event {
    public String helloText;
}

Create a Zirk to Send the Custom Event

Now that we have the custom event created, we can create a Zirk which will send this event by publishing it.

  1. Create a new java class file called HelloWorld in the same java directory.
  2. Import Bezirk and BezirkMiddleware classes and create a main method. 
    import com.bezirk.middleware.Bezirk;
import com.bezirk.middleware.java.proxy.BezirkMiddleware;

public class HelloWorld {
    public static void main(String args[]) {

    }
}
  3. In the main method, initialize Bezirk and get a Bezirk instance by calling BezirkMiddleware's initialize() and registerZirk(String zirkName) methods. The zirkName is just a human readable name for this Zirk. 
    BezirkMiddleware.initialize();
Bezirk bezirk = BezirkMiddleware.registerZirk("Hello World Zirk");
  4. Instantiate the HelloWorldEvent that we created earlier and set a custom text for the helloText variable.
    HelloWorldEvent helloWorldEvent = new HelloWorldEvent();
helloWorldEvent.helloText = "Hi world, how are you!";
  5. Finally, publish the event and confirm with a print message.
    bezirk.sendEvent(helloWorldEvent);
System.out.println("Published Hello World Message: " + helloWorldEvent.helloText);
    6. That's it! Click below to see the whole code.

    Show HelloWorld.java 

    import com.bezirk.middleware.Bezirk;
    import com.bezirk.middleware.java.proxy.BezirkMiddleware;

    public class HelloWorld {

        public static void main(String args[]) {
            //Initialize middleware and register this Zirk to get a Bezirk instance
            BezirkMiddleware.initialize();
            Bezirk bezirk = BezirkMiddleware.registerZirk("Hello World Zirk");

            //Create a new event and publish it
            HelloWorldEvent helloWorldEvent = new HelloWorldEvent();
            helloWorldEvent.helloText = "Hi world, how are you!";
            bezirk.sendEvent(helloWorldEvent);
            System.out.println("Published Hello World Message: " + helloWorldEvent.helloText);
        }
    }

Run the Example

Now we can run the HelloWorld program (right click the HelloWorld java file and select Run 'HelloWorld.main()').

You may see a pop up asking you to choose your active network for sending and receiving messages:

If so, select your active network and click "OK". If you're not sure which to select, try the first one and then the others if that doesn't work. If you don't see this popup, don't worry; that means Bezirk found the active network to use automatically.

You should see the confirmation output:

Great job! Now that you know how to publish events, check out the next tutorial on more in-depth Zirk communication, including how to subscribe to and receive events and do useful things with the data.
