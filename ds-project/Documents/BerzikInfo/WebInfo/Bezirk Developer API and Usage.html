API and Usage

Getting the API

We use Gradle for fetching the API and managing dependencies. If you're not familiar with Gradle, please see Project Setup for step-by-step instructions on getting started. Click below for the Gradle info to get the API:

Get the API

In the "repository" section of build.gradle, add: 
maven {
    url "https://bezirk.artifactoryonline.com/bezirk/libs-release-local"
}
In the "dependency" section, add: 
compile(group: 'com.bezirk', name: 'java-zirk-proxy', version: '3.0.2-3')
If you're developing for Android devices, please see Project Setup for Android.

Usage at a Glance

The following is a quick glance at using the Bezirk API. For detailed instructions, check out the step-by-step tutorials, starting with Hello World and Pub-Sub.

To initialize the middleware and get an instance of the Bezirk API: 
BezirkMiddleware.initialize();
Bezirk bezirk = BezirkMiddleware.registerZirk("Hello World Zirk");
info_outline Key Term: What is a Zirk?
A Zirk is any software that uses the Bezirk middleware to communicate. Basically, any time you use the Bezirk API, you are creating a Zirk.

To create an Event:

To create an Event for sending to other Zirks in order to communicate and interoperate, subclass Event and add any custom variables needed: 
public class HelloWorldEvent extends Event {
    public String helloMessage = "Hi World, how are you?";
}

To publish an Event: 
bezirk.sendEvent(new HelloWorldEvent());

To receive an Event:

Receiving events involves 3 short steps:

1. Create an EventSet, which is just a set of events. Since we have only one event right now, it's just a set of one. Bezirk lets you subscribe to event sets because later, there may be many related events to subscribe to and it'd be a hassle to do so one by one. 

EventSet eventSet = new EventSet(HelloWorldEvent);
2. For the eventSet, set an EventReceiver to process the event when it's received. 
eventSet.setEventReceiver(new EventSet.EventReceiver() { 
    @Override
    public void receiveEvent(Event event, ZirkEndPoint sender) {
        //Check if this event is a HelloWorldEvent
        if (event instanceof HelloWorldEvent) {
            //If it is, do something with the event
            HelloWorldEvent helloWorldEvent = (HelloWorldEvent) event;
            System.out.println("Received the following message: " + helloWorldEvent.helloMessage);
        }
    }
});
3. Subscribe to the event set. 
bezirk.subscribe(eventSet);

To Send an Event to a Specific Zirk:

Notice that the 
receiveEvent(Event event, ZirkEndPoint sender)
contains a ZirkEndPoint, which is a handle that allows you to send events directly to a Zirk. Simply pass it as a parameter in sendEvent(); for example, if you want to send a reply event back to the sender: 
bezirk.sendEvent(sender, someEventJustForTheSender);
